**命题一** — 感知是力量的基础  
力量的结构性差异根植于感知访问深层感质的方式，不是文化建构。

**命题二** — 力量不等于等级  
主要与主导是功能位差。任何将二者折叠为强弱关系的语言是滑义行为。

化学与语言发动，科技与音乐持有。两者互为结构条件。

**命题四** — 四通道，一目标  
感知的采样结构发生相变——多层同时在场，存在感知到自身平时无法抵达的频率层。
